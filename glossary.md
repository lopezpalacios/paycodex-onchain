# Glossary — DLT/EVM cash mgmt

## A

- **AA** — Account Abstraction. See [[concepts/account-abstraction]] · [[standards/erc-4337]]
- **Aave** — DeFi lending protocol
- **AMM** — Automated Market Maker (Uniswap, Curve)
- **APY** — Annual Percentage Yield

## B

- **Base** — Coinbase L2. See [[platforms/base]]
- **Besu** — Hyperledger Java EVM client. See [[platforms/besu]]
- **BUIDL** — BlackRock tokenized USD MMF. See [[cash-legs/tokenized-mmf]]
- **Bytecode** — compiled EVM instructions

## C

- **CASP** — Crypto-Asset Service Provider (MiCA). See [[compliance/mica-eu]]
- **CBDC** — Central Bank Digital Currency. See [[cash-legs/wholesale-cbdc]] · [[cash-legs/retail-cbdc]]
- **CCIP** — Chainlink Cross-Chain Interoperability Protocol. See [[concepts/ccip]]
- **CCTP** — Circle Cross-Chain Transfer Protocol. See [[concepts/circle-cctp]]
- **CEI** — Checks-Effects-Interactions. See [[concepts/checks-effects-interactions]]
- **Chainalysis** — chain analytics. See [[concepts/chainalysis]]

## D

- **Diamond** — modular contract pattern. See [[standards/erc-2535]]
- **DLT** — Distributed Ledger Technology. See [[01-dlt-fundamentals]]
- **DCHF** — Sygnum tokenized CHF deposit. See [[cash-legs/tokenized-deposit]]
- **DvP** — Delivery vs Payment (atomic security-cash). See [[architecture/atomic-dvp-pattern]]

## E

- **EAS** — Ethereum Attestation Service. See [[concepts/eas]]
- **EIP** — Ethereum Improvement Proposal
- **ENS** — Ethereum Name Service. See [[concepts/ens]]
- **EOA** — Externally Owned Account (vs smart contract account)
- **ERC** — Ethereum Request for Comment (token standard family). See [[standards/]]
- **EVM** — Ethereum Virtual Machine. See [[02-evm-stack]]
- **EURCV** — SocGen FORGE tokenized EUR deposit. See [[cash-legs/tokenized-deposit]]
- **EURC** — Circle euro stablecoin. See [[cash-legs/stablecoin-public]]

## F

- **FATF R16** — Travel Rule recommendation
- **Fireblocks** — MPC custody vendor. See [[concepts/mpc-custody]]
- **FORGE** — SocGen DLT subsidiary issuing EURCV
- **Foundry** — Solidity toolkit. See [[concepts/foundry]]

## G

- **Gas** — EVM execution cost unit
- **gpi** — already in cashmgmt glossary
- **Gnosis Safe** — most-used multisig. See [[concepts/multisig-gnosis-safe]]

## H

- **Hardhat** — JS Solidity framework. See [[concepts/hardhat]]
- **HSM** — Hardware Security Module. See [[concepts/hsm-custody]]
- **HTLC** — Hash Time Locked Contract (cross-chain atomic swaps)

## I

- **IVMS-101** — Inter-VASP Messaging Standard (Travel Rule data format)

## J

- **JPMD** — JPMorgan tokenized USD deposit (on Onyx). See [[cash-legs/tokenized-deposit]]

## L

- **L1 / L2** — Layer 1 (Ethereum) / Layer 2 (rollups). See [[platforms/]]
- **LayerZero** — cross-chain messaging. See [[concepts/layerzero]]

## M

- **mBridge** — already in cashmgmt glossary; multi-CBDC platform
- **Merkle proof** — efficient inclusion proof for set membership
- **MEV** — Maximal Extractable Value (front-running, sandwich)
- **MiCA** — already in cashmgmt glossary; EU crypto reg
- **MPC** — Multi-Party Computation custody. See [[concepts/mpc-custody]]

## N

- **Notabene** — Travel Rule SDK vendor

## O

- **OFAC SDN** — already in cashmgmt glossary; smart contracts can be sanctioned. See [[concepts/tornado-cash-precedent]]
- **Onyx** — JPMorgan permissioned chain platform
- **OpenZeppelin** — audited Solidity library. See [[concepts/openzeppelin]]
- **ONCHAINID** — T-REX identity standard. See [[concepts/onchainid]]

## P

- **Paymaster** — ERC-4337 component sponsoring gas. See [[architecture/gasless-paymaster-pattern]]
- **PoR** — Proof of Reserves. See [[concepts/proof-of-reserves]]
- **PoS** — Proof of Stake (Ethereum consensus post-Merge)
- **Polygon** — sidechain + L2 family. See [[platforms/polygon-pos]] · [[platforms/polygon-zkevm]]

## Q

- **Quorum** — ConsenSys permissioned Geth fork. See [[platforms/quorum]]

## R

- **Rebasing** — token supply auto-adjusts
- **Reentrancy** — recursive call attack. See [[concepts/reentrancy]]
- **Rollup** — L2 type (optimistic vs ZK)

## S

- **Safe** — Gnosis Safe multisig
- **SDX** — SIX Digital Exchange (Swiss DLT venue)
- **Sygnum** — Swiss crypto-bank issuing DCHF
- **Solidity** — main EVM language. See [[concepts/solidity]]

## T

- **T-REX** — ERC-3643 compliance framework. See [[standards/erc-3643-trex]] · [[compliance/trex-erc-3643]]
- **The Graph** — decentralized indexing. See [[concepts/the-graph]]
- **Timelock** — delayed execution. See [[concepts/timelock]]
- **Tokeny** — leading T-REX vendor (LU)
- **TRP** — Travel Rule Protocol (TRUST). See [[compliance/travel-rule-onchain]]
- **TRISA** — open Travel Rule protocol
- **TRM Labs** — chain analytics

## U

- **UETR** — already in cashmgmt glossary; tracks SWIFT gpi
- **UUPS** — Universal Upgradeable Proxy Standard. See [[concepts/uups]]
- **USDC** — Circle USD stablecoin. See [[cash-legs/stablecoin-public]]
- **USDT** — Tether USD stablecoin. See [[cash-legs/stablecoin-public]]

## V

- **VASP** — Virtual Asset Service Provider (FATF term)
- **Vyper** — alternative EVM language

## W

- **Wormhole** — cross-chain messaging. See [[concepts/wormhole]]

## X

- **xERC20** — ERC-7281 cross-chain fungible. See [[standards/erc-7281]]

## Z

- **zkEVM** — ZK-proof EVM (zkSync, Polygon zkEVM, Scroll). See [[platforms/zksync]] · [[platforms/polygon-zkevm]]
- **ZK-KYC** — zero-knowledge KYC. See [[concepts/zk-kyc]]
