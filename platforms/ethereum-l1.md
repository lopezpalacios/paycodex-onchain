# Ethereum L1

Base chain. Highest security, highest cost.

| Property | Value |
|---|---|
| Block time | ~12s |
| Finality (probabilistic full) | ~13min (2 epochs) |
| TPS | ~15-30 |
| Cost (typical) | $1-50 / tx (ERC-20 transfer ~$5-20) |
| Consensus | PoS (post-Merge 2022) |

## Use for cash mgmt

- Final settlement layer for largest value tx
- Tokenized treasury / institutional money market funds (BUIDL is on Ethereum)
- Stablecoin issuance home (USDC, USDT both have L1 home + L2 deployments)

## When to use

- Settlement >$100K where finality + decentralization matter
- Bridge anchor for L2 deployments
- Avoid for high-frequency / micro tx
