# Hyperledger Besu

Open-source Java EVM client. Most popular permissioned EVM choice for banks.

## Key features

- IBFT 2.0 / QBFT consensus (Byzantine Fault Tolerant)
- Permissioning at node + account level
- Privacy via Tessera (private transactions)
- Full EVM compatibility — Solidity contracts deploy unchanged

## Use cases

- Bank consortium chain
- Wholesale CBDC pilots (some)
- Private settlement networks
- T-REX deployments
