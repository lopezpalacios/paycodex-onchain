# Base

Coinbase's L2 (optimistic rollup, OP Stack). Low cost, broad adoption, EVM-equivalent.

| Property | Value |
|---|---|
| Block time | ~2s |
| Finality (to L1) | ~7 days (challenge period) |
| TPS | ~100s |
| Cost | $0.001-0.05 / tx |
| Bridge | Native to L1, Circle CCTP for USDC |

## Why for cash mgmt

- Fastest-growing US/EU enterprise L2
- Cheap micro-payments
- USDC native (Circle preferred)
- Coinbase relationship — institutional onboarding easier
