# Polygon PoS

Sidechain with own validator set, EVM-compatible. Older than zkEVM.

| Property | Value |
|---|---|
| Block time | ~2s |
| Finality | ~256 blocks |
| Cost | extremely cheap |

## Use

- Long-running stablecoin liquidity outside Ethereum
- BlackRock BUIDL deployed here
- Enterprise pilots (JPM, Mastercard)
