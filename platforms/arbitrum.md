# Arbitrum

Largest L2 by TVL. Optimistic rollup, EVM-equivalent (Nitro).

| Property | Value |
|---|---|
| Block time | ~250ms |
| Finality (to L1) | ~7 days |
| TPS | ~100s, can scale to 1000s |
| Cost | $0.001-0.10 / tx |

## Use

- DeFi-heavy, biggest stablecoin liquidity outside L1
- USDC + USDT native
- Treasury yield strategies (ERC-4626 vaults plentiful)
