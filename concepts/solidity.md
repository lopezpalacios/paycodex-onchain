# Solidity

Statically typed compiled language for EVM. Most-used smart contract language.

## Versions

- 0.8.x current standard (overflow safe by default since 0.8.0)
- Pin pragma: `pragma solidity ^0.8.20;`

## Cash mgmt usage

- All ERC standards implemented in Solidity
- OpenZeppelin contracts (audited library) — use as base, extend
- Hardhat (default) / Foundry for build + test
