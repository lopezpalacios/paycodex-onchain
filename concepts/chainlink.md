# Chainlink

Largest decentralized oracle network. Plus CCIP cross-chain messaging.

## Products relevant to cash mgmt

- **Data Feeds** — price (FX, rates, commodities)
- **Proof of Reserve** — stablecoin reserves attestation
- **CCIP** — cross-chain messaging + token transfers
- **Functions** — call any HTTPS API from contract
- **VRF** — verifiable randomness (less relevant here)
- **Automation** — chain-time triggers (cron-like for contracts)
