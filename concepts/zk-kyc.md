# ZK-KYC

Prove KYC compliance without revealing identity. Zero-knowledge proof attests "this wallet passed KYC at provider X" without exposing KYC data.

## Use

- Permissioned compliance with public-chain-style privacy
- Travel Rule compliance via zk-proofs
- Selective disclosure (jurisdiction yes, exact name no)

## Status

- Active research, several pilots (Polygon ID, Soulbound research)
- Production-grade still emerging
