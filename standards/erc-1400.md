# ERC-1400 — Security Token

Standard for tokenized securities with regulatory hooks.

## Components

- **Partitions** — token portions with different rules (e.g., tranches, lock-ups)
- **Transfer restrictions** — `canTransfer` predicate enforced on-chain
- **Document attachment** — IPFS hashes of legal docs
- **Operator pattern** — controllers (trustees, transfer agents) can move tokens

## Status

- Pioneer security token standard (~2018)
- Largely superseded by ERC-3643 (T-REX) for compliance + identity
- Some legacy issuances still on ERC-1400
