# ERC-4626 — Tokenized Vault

Standard for yield-bearing vaults. Deposit underlying, receive shares; redeem shares, get underlying + yield.

## Cash mgmt use

- **Tokenized MMF** — BlackRock BUIDL = essentially ERC-4626-style
- **Treasury yield deployment** — deposit USDC into vault, get yield
- **Liquidity pool participation**
- **Repo collateral pool**

## Interface

```solidity
function asset() external view returns (address); // underlying ERC-20
function totalAssets() external view returns (uint256);
function deposit(uint256 assets, address receiver) external returns (uint256 shares);
function mint(uint256 shares, address receiver) external returns (uint256 assets);
function withdraw(uint256 assets, address receiver, address owner) external returns (uint256 shares);
function redeem(uint256 shares, address receiver, address owner) external returns (uint256 assets);

function convertToShares(uint256 assets) external view returns (uint256 shares);
function convertToAssets(uint256 shares) external view returns (uint256 assets);
```

## Pattern

- Treasury holds USDC → deposits to MMF vault → receives BUIDL shares
- Vault yields ~5% (T-bill rate) → share price increases against USDC
- Treasury redeems → gets USDC back + yield in single tx
