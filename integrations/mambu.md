# Mambu + DLT

Cloud-native, API-first. Easy to wrap DLT operations as a Mambu product.

## Pattern

- Mambu account = mirror to DLT wallet
- Mambu API call → tokenization service → chain tx
- Webhook on chain event → Mambu balance update

## Examples

- Several neobanks on Mambu have DLT-touching products
