# ADR 0006 — Tokenization platform: build vs Tokeny

**Status**: Proposed
**Date**: 2026-04-25

## Decision

**Buy Tokeny T-REX framework as foundation. Build custom on top (issuance manager, ops console, integration with core banking).**

## Reasoning

- Tokeny operates ERC-3643 reference + several major issuances
- Cheaper than full build
- Customizable enough
- Vendor lock manageable (open standard)
